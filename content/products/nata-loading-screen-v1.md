# Nata Loadingscreen v1 - FiveM Script

Nata Loadingscreen v1 adalah script loading screen modern untuk server FiveM yang memberikan tampilan profesional dan menarik saat player pertama kali masuk ke server.

## Fitur Utama
- Background video atau gambar (custom)
- Music player (YouTube atau file lokal)
- Informasi server (nama, deskripsi, rules)
- Progress loading bar yang smooth
- Integrasi social media (Discord, Instagram, website)
- Desain UI modern, clean, dan responsif
- Performa ringan dan optimal

## Teknologi
- HTML5
- CSS3
- JavaScript

## Struktur
Struktur file disusun rapi dan mudah dipahami, sehingga memudahkan proses editing dan pengembangan lebih lanjut.
